# 🤖 AI Chatbot

## 📖 Overview

The **AI Chatbot** is a simple web application built using **HTML, CSS, and JavaScript**. It provides an interactive chat interface where users can communicate with a virtual assistant and receive instant responses. The project is designed to demonstrate basic chatbot functionality and frontend web development concepts.

## 🚀 Features

* Interactive chat interface
* Send and receive messages
* Instant chatbot responses
* Clean and modern design
* Responsive layout
* Easy-to-use user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `chatbot.html` in your browser.
4. Type a message and click **Send**.
5. View the chatbot response instantly.

## 💡 Example Output

```text
User: Hi
Bot: Hello! 👋 How are you today?

User: What is JavaScript?
Bot: JavaScript is a programming language used to create interactive web pages.
```
